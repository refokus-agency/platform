# Code of Conduct

This project adopts the **[Contributor Covenant, version 3.0](https://www.contributor-covenant.org/version/3/0/code_of_conduct/)** as its code of conduct.

The full text (Community Pledge, Community Standards, Enforcement Guidelines, and attribution) is maintained upstream at contributor-covenant.org and incorporated here by reference.

## Scope

The code of conduct applies within all community spaces for this project — issues, pull requests, discussions, and any direct communication tied to the project — and when an individual is officially representing the project in public spaces.

## Reporting

Concerns about behavior in this project's community should be reported privately to:

**tomas@refokus.com**

Reports will be reviewed and investigated promptly and fairly. The reporter's identity will be kept confidential.

For security vulnerabilities (not behavior), see [`SECURITY.md`](SECURITY.md) instead.

## Enforcement

Community leaders will follow the [Enforcement Guidelines](https://www.contributor-covenant.org/version/3/0/code_of_conduct/#enforcement-guidelines) from the Contributor Covenant when determining the consequences of any action deemed in violation.
